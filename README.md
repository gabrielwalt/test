# AEM Experience Modernization Console – Mock

A standalone HTML mock of the **Adobe Experience Manager (AEM) Experience Modernization (ExMod) Console** UI.

## What it is

A single-file prototype of the ExMod console shell: header, left nav, Task Progress panel, chat panel, and preview panel. Uses Adobe Spectrum design tokens (light theme).

## Usage

Open `index.html` in a browser. No build step or server required.

## Structure

- **index.html** – Fully self-contained mock (embedded styles, inline SVGs, no external assets)
- **AGENTS.md** – Agent instructions for working on this project
